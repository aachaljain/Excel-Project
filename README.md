# Excel-Project
This project focuses on analyzing retail sales data to identify trends, customer behavior, and business performance. Using tools such as Excel and data visualization techniques, the dataset was cleaned, transformed, and analyzed to generate meaningful insights. Key metrics such as total sales, order trends, customer demographics, and product performance were evaluated. Interactive dashboards and visual reports were created to help stakeholders understand sales patterns and make data-driven decisions. The project demonstrates practical skills in data cleaning, analysis, and visualization.
